# AI ETL Data Quality API

FastAPI-based backend service for CSV ingestion, ETL processing,
automated data quality checks, and reporting.

## Features
- CSV upload API
- Raw and cleaned data storage
- Data quality validation
- REST-based quality reports

## Tech Stack
Python, FastAPI, Pandas, SQLAlchemy, SQLite, Pytest
